# 🎓 MyGrad

O **MyGrad** é um Progressive Web App (PWA) desenvolvido para a gestão e controle da rotina acadêmica de estudantes universitários. O aplicativo nasceu da necessidade de substituir planilhas complexas por uma interface ágil, visual e intuitiva, permitindo o acompanhamento de disciplinas, notas, faltas e prazos diretamente do celular ou computador.

## 🛠️ Funcionamento e Recursos

- **Controle de Faltas:** Permite o cadastro de disciplinas com seus respectivos limites de faltas. O app conta com botões rápidos de incremento/decremento e uma barra de progresso visual com gradiente dinâmico de cores, emitindo alertas automáticos ao atingir 80% e 100% do limite permitido.
- **Gestão de Notas & Desempenho:** Lançamento de avaliações (provas, trabalhos e projetos) por disciplina, com cálculo automatizado do Coeficiente de Rendimento (CR) do período acadêmico e exibição do status do aluno.
- **Calendário de Eventos:** Central de cadastro de eventos para organizar os compromissos acadêmicos do semestre.
- **Dashboard Centralizado:** Painel geral que reúne o resumo do desempenho semestral, gráficos informativos e uma listagem com a contagem regressiva para os próximos prazos e compromissos da semana.
- **Arquitetura PWA & Modo Offline:** Graças à implementação de *Service Workers*, o aplicativo realiza o cache dos arquivos essenciais, permitindo inicialização instantânea e funcionamento completo mesmo sem conexão com a internet.
- **Privacidade & Backup Local:** O banco de dados roda de forma 100% exclusiva no **LocalStorage** do dispositivo do usuário. O sistema também conta com uma ferramenta de exportação e importação de dados via arquivos `.json`, permitindo o backup seguro ou sincronização entre múltiplos aparelhos.
