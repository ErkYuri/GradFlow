# 🎓 MyGrad

O **MyGrad** é um Progressive Web App (PWA) desenvolvido para auxiliar estudantes no gerenciamento da rotina acadêmica, permitindo acompanhar faltas, notas, desempenho e compromissos de forma simples, visual e organizada.

---

## 🔗 Acesso

O aplicativo está disponível para uso através da Vercel:
👉 [Acesse o MyGrad aqui](https://my-grad.vercel.app/)

---

## 🛠️ Funcionamento e Recursos

- **Controle de Faltas:** Permite o cadastro de disciplinas com seus respectivos limites de faltas.
- **Gestão de Notas & Desempenho:** Lançamento de avaliações e atividades por disciplina, com cálculo automático do Coeficiente de Rendimento e exibição do status do aluno.
- **Calendário de Eventos:** Central de cadastro de eventos para organizar os compromissos acadêmicos do semestre.
- **Dashboard Centralizado:** Painel geral que reúne o resumo do desempenho e andamento semestral.
- **Arquitetura PWA & Modo Offline:** O aplicativo realiza o cache dos arquivos essenciais, permitindo inicialização instantânea e funcionamento completo mesmo sem conexão com a internet.
- **Privacidade & Backup Local:** O banco de dados roda no **LocalStorage** do dispositivo do usuário, além de uma ferramenta de exportação e importação de dados via arquivos para backup entre dispositivos.
